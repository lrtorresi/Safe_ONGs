Mobile

1º - instalar o EXPO
(npm install -g expo-cli)
Comando para ver se esta funcionando (expo-h)
Se der erro, rodar o comando Powershell (Set-ExecutionPolicy Bypass)

2º - Criar o projeto Mobile
Comando - (expo init mobile) selecionar template Blank

3º - startar web-mobile (celular)
Comando - (yarn start)

4º - Instalar a navegação do expo 
Comando - (npm install @react-navigation/native)

Comando - (expo install react-native-gesture-handler react-native-reanimated react-native-screens react-native-safe-area-context @react-native-community/masked-view)

Tipo de navegação: 
Comando - (npm install @react-navigation/stack)

5º instalar um pacote de design/layout
COmando - (expo install expo-constants)

6º chamar função de e-mail
Comando - (expo install expo-mail-composer)

7º conectar na API (cliente HTTP)
Comando - (npm install axios)

8º Pacote para formatação de caracteres ($)
Comando (npm install intl)
